# Node Mapping Notation

# Examples

**example1.nmn**
```yaml
version: 0.1
add:
- "example2.nmn"
nodes:
- id: 'node1'
  name: 'NodeExample'
```

**example2.nmn**
```yaml
version: 0.1
nodes:
- id: 'node2'
  name: 'NodeExample'
- id: 'node3'
  name: 'NodeExample'
```
